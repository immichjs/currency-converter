## Para executar em sua máquina local
```
- git clone https://github.com/immichjs/currency-converter.git

- cd currency-converter

- npm i
```

### Para iniciar um servidor de desenvolvimento
```
npm run serve
```

### Compilar projeto para produção
```
npm run build
```
